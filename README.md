# Personal Portfolio

This is a simple Flask application.

## Setup Instructions

Follow these steps to set up and run the Flask app.

### Prerequisites

- Python 3
- Virtual environment (optional but recommended)

### 1. Clone the Repository

```bash
git clone <your-repo-url>
cd portfilio
```
### 2. Start Flask App

python3 -m venv .venv
.venv\Scripts\activate

pip install flask
flask --app app run
